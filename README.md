# Vyasa landing

Marketing site and waitlist (Next.js).

## Development

```bash
npm install
npm run dev
```

For waitlist and backend behavior, copy `.env.example` to `.env.local` and set the variables described there.

## Deploy

Deploy with [Vercel](https://vercel.com) or any Node host that supports Next.js. Configure the same environment variables as in production on your host.
